Initial Review

  1. Click on the submission link provided.
  2. Match the output with the provided test cases.
  3. If correct: Mark the Google Sheet cell as dark green.
  4. If incorrect: Mark the cell red and comment on the error.

Providing Feedback

  • If the submission is correct, use the feedback template to respond.
  • After sending feedback, change the sheet cell from dark green to light green.

Cross-Checking Errors

  1. Once the initial review is complete, another reviewer must check all red-marked cells.
  2. If the cross-checker confirms the error, they should reply with "Correct".
  3. If the error is incorrectly marked or needs further clarification, the cross-checker should provide an appropriate reply.
  4. Change the cross-checked cells to light blue.

Final Recheck

Re-evaluate all light blue cells for accuracy. If the error was correctly identified, send the final feedback and update the comment to "Given".
